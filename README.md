# 🍕 Pizza Sales Analysis | SQL + Power BI

## 📌 Project Overview
This project focuses on analyzing pizza sales data using SQL and Power BI to uncover meaningful business insights related to revenue, customer ordering behavior, sales performance, and product trends.

The project includes:
- Data cleaning and transformation using Power Query
- SQL-based sales analysis
- DAX KPI calculations
- Interactive Power BI dashboards
- Business insights and trend analysis

---

## 📈 Key Insights
- Highest orders were recorded on weekends, especially Friday and Saturday evenings.
- July and January had the highest order volumes.
- Large-size pizzas contributed the highest sales.
- Classic pizza category generated maximum revenue and orders.
- Thai Chicken Pizza generated the highest revenue.
- Brie Carre Pizza had the lowest sales performance.
